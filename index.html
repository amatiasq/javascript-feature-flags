Javascript Feature Flags Demo

A quick demo of Javascript Feature Flags code.

Flags That Are Set on this Page:

These flags have been set via cookies, URL query strings, URL fragments and/or the domain name.

Example URLs with Flags

  • #ff,Jimi
  • #ff,Elvin,Ringo
  • &ff=Eddie,Sonny
  • &ff=Jimmy&ff=Cecil
